Share My ORCID iD

A simple way to collect authenticated ORCID iDs

Collect authenticated ORCID iDs, just by sending a link!

Want to get ORCID iDs from attendees at your next conference or event? Or maybe from everyone in your research group/lab? Share My iD makes it simple!

  1. Create a new iD collection (anyone with an ORCID iD can create a new collection!)
  2. Share the collection URL with people whose ORCID iDs you want to get (or display the page on a laptop/tablet at your event).
  3. Visit the collection URL to see and download iDs you've collected.

Not sure if everyone you want to get an ORCID iD from actually has one? No problem! Users can register for an iD and add it to your collection at the same time.

Why 'authenticated' ORCID iDs?

ORCID is all about sorting out who's who, so it's important that you get the right iD for each person. No room for typos here. Share my iD collects authenticated iDs, which means that each user's iD is sent straight from ORCID after they log into their ORCID account. You always get the correct iD for the person who logged in. Learn more about authenticated iDs

Want to do more with ORCID?

Share my iD was built by the fine folks at ORCID, but it's powered by APIs available to both ORCID member organizations and freely to the public. You can build your own ORCID-fuled applications. Learn more about ORCID APIs
